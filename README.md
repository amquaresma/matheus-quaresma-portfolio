Matheus Quaresma — Portfólio Pessoal
 Este repositório é o meu portfólio pessoal — um espaço onde reúno minha trajetória, projetos, habilidades e conquistas de forma visual e interativa.
O site foi desenvolvido completamente do zero, como uma forma de demonstrar domínio sobre os fundamentos do desenvolvimento web: HTML semântico, CSS moderno e JavaScript puro.

Por que criei este portfólio
Acredito que um portfólio bem feito diz mais sobre um desenvolvedor do que qualquer currículo. Queria criar algo que fosse ao mesmo tempo técnico e pessoal — que mostrasse não só o que sei fazer, mas como penso e como me apresento profissionalmente.
Cada detalhe do design foi escolhido com intenção: a tipografia Orbitron para passar uma identidade tech, a paleta escura com indigo e teal para transmitir sofisticação, e as animações sutis para dar vida à experiência sem poluir o conteúdo.

Sobre mim
Estou cursando Engenharia da Computação no SENAC e Ciência de Dados na FATEC, além de ter concluído o curso técnico em Desenvolvimento de Sistemas pela ETEC. Em 2025, conquistei medalhas em olimpíadas nacionais de tecnologia e programação, incluindo ouro na ONC/OBGP, prata na OBG, bronze na TNB e menção honrosa na OBT e na Maratona Tech.
Tenho experiência prática com JavaScript, TypeScript, React, React Native, Firebase, MySQL, PHP, C# e Python — e estou sempre buscando aprender mais.

Tecnologias do projeto

HTML5, CSS3 e JavaScript — sem frameworks, sem dependências
Google Fonts: Orbitron, DM Sans e JetBrains Mono
Design system próprio com custom properties CSS
Animações nativas: partículas, scan line, scroll reveal e splash screen
